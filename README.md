# How to use this repository
1. Fork this repository.
1. Open https://github.com/settings/personal-access-tokens
1. Create a fine-grained personal access token with read and write access to the "Followers" user permissions.
1. Open https://docs.github.com/en/actions/how-tos/write-workflows/choose-what-workflows-do/use-secrets
1. Save the fine-grained personal access token with the name `GH_TOKEN`
